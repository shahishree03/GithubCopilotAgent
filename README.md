# My Copilot Plugin

A custom plugin for GitHub Copilot that adds new capabilities.

## Features
- Hello World command

## Installation
1. Clone this repository
2. Run `npm install`
3. Run `npm run build`
4. Install in your Copilot agent

## Development
- `npm run build` - Build the plugin
- `npm run dev` - Watch for changes and rebuild
